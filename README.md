# 🤖 AI Customer Support Triage (OpenEnv)

## 🚀 Live Demo
👉 https://huggingface.co/spaces/piyushk12345/ai-support-openenv

---

## 📌 Project Overview
This project simulates a real-world customer support system where an AI agent classifies user queries into categories like:

- Refund
- Complaint
- General Query

The system provides reward-based feedback, making it suitable for training and evaluating AI agents.

---

## 🧠 Features
- OpenEnv compliant environment
- Multiple tasks (easy → hard)
- Reward-based evaluation
- Dockerized deployment
- Live on HuggingFace

---

## ⚙️ Tech Stack
- Python
- OpenEnv
- Flask
- Docker
- HuggingFace Spaces

---

## 🧪 How It Works
1. User sends a message  
2. System classifies it  
3. Reward is given based on accuracy
